DESCRIÇÃO: O nosso trabalho se resume em um jogo 2D de tiro em terceira pessoa nomeado "WILD WEST SHOWDOWN", que é baseado no jogo 2D "Sunset Riders" 
de 1991 desenovolvido pela Konami.

MODO DE COMPILAÇÃO E EXECUÇÃO: Devido ao uso de um arquivo "Makefile", tanto a compilação quanto a execução se resumem em abrir a pasta que contem os 
arquivos no terminal e digitar na linha de código "make excutar". Mas, para isso, deve-se estar instalado em seu sistema o peracional apenas as bibliotecas 
usuais OpenGL, FreeGlut e SOIL.

COMO JOGAR: Durante o jogo, controlaremos apenas o personagem principal que é um cowboy. Para operá-lo, utilizamos a tecla "a" para andar para a esquerda,
a tecla "w" para andar para cima, a tecla "s" para andar para baxo, a tecla "d" para andar para a direita e "enter" para realizar um disparo com o personagem.
Além disso, utiliza-se o click esquerdo do mouse para acessar outros menus e "esc" para fechar o jogo totalmente. O objetivo é eliminar os inimigos para pontuar
e, assim, passar para os próximos níveis e, por fim, disputar contra o boss final, evitando os ataques dos inimigos que, caso acertem o cowboy, um dano será receido
e a quantidade de vida diminuirá.
